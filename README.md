# Team32

This repo includes Operation: EarthCore game source code, documentation, reports, interface artifacts, and setup instructions 

## Local Setup Instructions

 - Open the [`releases`](releases) folder on GitHub
 - Download the desired build corresponding to your OS
 - Unzip the archive with any file-archiver program (e.g. 7-Zip, WinRAR)
 - Open a folder with unzipped files
 - Run `TheMoleProject2D.exe` (Windows) or `TheGame.x86_64` (Linux) to start the game
